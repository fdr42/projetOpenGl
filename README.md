Projet inspiré par snake en c avec opengl
=========================

Comment compiler le projet ?
-----------------------

1. Sur Mac : make apple

2. Sur linux : make


Comment lancer le projet ?
---------------------

Pour lancer le projet faire : ./snake


Comment jouer ?
-------------

Le but du jeu est d'arriver à la fin du plateau de jeu mais des
obstacles vous barrent la route. Il faut les detruire en lançant des
projectiles ou en foncant dedans. Vous pouvez recuperer des boules
présentes sur le plateau. 

1. La touche "Q" permet de tourner à gauche.

2. La touche "D" permet de tourner à droite.

3. La touche "A" permet de tirer des projectiles.
